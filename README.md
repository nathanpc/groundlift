# GroundLift

An AirDrop alternative.

## Compilation Requirements

In order to compile this project you're required to install a couple of
packages. In case you're using Ubuntu just run the following command:

```bash
sudo apt install build-essential pkg-config bear libavahi-core-dev libavahi-client-dev libavahi-common-dev
```
